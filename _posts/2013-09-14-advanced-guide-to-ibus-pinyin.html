如何打开IBUS拼音输入法首选项？

在终端运行下述命令即可打开首选项窗口：

{% highlight bash %} /usr/lib/ibus-pinyin/ibus-setup-pinyin {% endhighlight %}
如何修改水平候选词条为垂直候选词条？

打开首选项窗口 → preferences → general → orientation of candidate，选择vertical即可。

在IBUS输入法中如何输入繁体字？

打开首选项窗口 → preferences -> general -chinese，选择traditional。该设置需要注销并重新登入。

IBUS拼音输入法如何修改候选词翻页快捷键？

ibus-pinyin默认的翻页快捷键是 pageup,pagedown。

打开首选项窗口 → preferences → pinyin mode → input custom，根据需要选择即可。具体含义如下：

  • [-][=] flip page，允许按-=进行翻页
  • [,][.] flip page，允许按,.进行翻页
ibus拼音输入法如何增加“候选词显示数”？

打开首选项窗口 → preferences → general → number of candicates

ibus拼音输入法如何输入左箭头？

在搜狗输入法中，输入zjt就可以快速输入左键头符号。IBUS输入法默认没有提供该功能。但是我们可以通过自定义短语来实现。通过自定义短语，还可以实现快速输入右箭头、上箭头、下箭头等。

  1. 打开 附件 → 字符映射表 ，选择 菜单 → 搜索 → 查找，在搜索输入框中输入 “leftwards arrow”并进行搜索。然后双击左箭头符号，它会出现在窗口下方要复制的文本输入框中，然后点击输入框右边的复制按钮。
  2. 打开IBUS输入法首选项窗口 → preferences → dictionary → edit custom phrases。在打开的phrases.txt文件的末尾输入 zjt=，然后按 ctrl+v，就把字符映射表中复制的符号粘贴过来了。
  3. 注销并重新登入
i模式

IBUS拼音输入法处于中文模式时，直接键入字母i即可进入i模式

i模式下可以进行下述特殊输入功能

  • gz.打印稿纸
  • hh.画画
  • js.计算模式
  • rq.输入日期
  • sj.输入时间
  • sz.掷骰子
  • xz.查询星座
  • zf.打印字符

在这里我不得不吐槽下i模式。i模式下的多数功能都是鸡肋！比如打印稿纸，画出来的图到底是什么意思？用户在什么场景下会使用该功能？我愣是没有想出来！再说画画功能。该功能本是一个及其值得期待的功能，搜狗输入法中的字符画让我爱不释手，可是ibus的画画功能就只能说聊胜于无了，默认提供的字符画没创意还让人看不懂。i模式下我唯一觉得有实用价值的就是查询星座了。输入ixz就可以查询任意具体一天对应的星座，比如ixz2011-01-27

v模式

IBUS拼音输入法处于中文模式时，直接键入字母v可以临时进入英文输入模式。在这个临时的英文输入模式里，按空格键确认输入并返回中文输入模式，按ESC返回中文模式。v模式避免了用户频繁切换输入模式，能显著提升输入速度。
