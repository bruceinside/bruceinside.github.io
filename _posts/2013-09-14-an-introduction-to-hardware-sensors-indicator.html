今天给大家推荐一款支持ubuntu 应用指示器的温度监控软件 - 硬件温度指示器，该软件支持ubuntu 11.10 和 12.04，是由澳大利亚友人 Alex Murray 开发的，主页在这里。

功能
  • 可以监控CPU，显卡，硬盘温度，甚至还可以监控Nvidia显卡风扇速度！
  • 支持在顶部面板中显示传感器图标、名称、温度，这个功能非常实用，无需任何操作，硬件温度映入眼帘。
  • 支持随机启动。
  • 硬件温度单位可以选择摄氏度还是华氏度。

不足
  • 没有中文版本
  • 不支持在顶部面板中同时显示多个传感器信息，这不能不算一个遗憾。
安装
{% highlight bash %} sudo apt-add-repository ppa:alexmurray/indicator-sensors sudo apt-get update sudo apt-get install indicator-sensors {% endhighlight %}
