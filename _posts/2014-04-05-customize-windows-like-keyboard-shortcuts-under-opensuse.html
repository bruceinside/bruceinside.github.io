熟悉windows系统的人，大概都是熟悉那些和Windows 徽标键相关的快捷键的，例如按下Win + D组合键，可以快速显示桌面。那在openSUSE系统中，是否也可以实现同样的功能呢？答案是肯定的。 下面我带领大家设置几个非常常见的快捷键，仅以抛砖引玉。

Win + D（显示桌面）
  1. 打开系统设置窗口
  2. 选择快捷方式和手势
  3. 点击左边栏中的全局键盘快捷键
  4. KDE组件选择 “kwin”
  5. 在搜索里输入“显示桌面”
  6. 点击搜索结果列表中的“显示桌面”
  7. 选择自定义，设置快捷键为 Win + D
Win + E（打开 Dolphin 文件管理器）
  1. 打开系统设置窗口
  2. 选择快捷方式和手势
  3. 点击左边栏中的自定义快捷键
  4. 在树状导航窗口中点击右键
  5. 选择新建 → 全局快捷键 → 命令/URL
  6. 把新增的条目重命名为“打开Dolphin文件管理器”
  7. 在右边的注释页签中输入“打开Dolphin文件管理器”
  8. 在触发器页签中设置快捷键为Win + E
  9. 在动作页签中输入命令: 
    dolphin
Win + F（搜索文件或文件夹）
  1. 运行命令 
    sudo zypper in kfind
     安装kfind程序
  2. 打开系统设置窗口
  3. 选择快捷方式和手势
  4. 点击左边栏中的自定义快捷键
  5. 在树状导航窗口中点击右键
  6. 选择新建 → 全局快捷键 → 命令/URL
  7. 把新增的条目重命名为“打开kfind”
  8. 在右边的注释页签中输入“打开kfind”
  9. 在触发器页签中设置快捷键为 Win + F
  10. 在动作页签中输入命令:  
    kfind
Win + L(锁定您的计算机)
  1. 打开系统设置窗口
  2. 选择快捷方式和手势
  3. 点击左边栏中的全局键盘快捷键
  4. KDE组件选择“KDE会话管理器”
  5. 在搜索里输入“lock session”或者 "锁住会话"
  6. 点击搜索结果列表中的“lock session”或者 "锁住会话"
  7. 选择自定义，设置快捷键为 Win + L

有点令人遗憾的是，Win键在openSUSE中被识别为Meta键，而单独的Meta键是不能作为快捷键的。也就是说，我们无法让openSUSE像windows那样，按一下Win键就打开开始菜单，至少在不修改键盘映射的情况下无法实现。
